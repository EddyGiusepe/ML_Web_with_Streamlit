# `Streamlit` para Série Temporal

Neste script vamos implementar uma interfaz simples para a visualização de dados. Esses dados são registros de marcas entre os anos $2000$ e $2017$.

Os dados usados são do portal de [dados.gov.br](https://dados.gov.br/home), estes dados são dados do `Anuário Estatístico de Propriedade Industrial`.


## Executando o script

streamlit run streamlit_app.py
